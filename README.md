# Documents

Documents is the file-type wrapper plugin for document artifact workflows.

This plugin is bundled into the Codex primary runtime.

## Included Skills

- `Documents`: create, edit, render, verify, and export editable Word and Google Docs document artifacts.

## Discoverability

Use this plugin for document-oriented terms from the file-type naming model: doc, docs, Word, Google Docs, Google Doc, memo, write, document, documents, and `.docx`.

## Source And Copy Flow

This plugin package is generated from `lib/artifacts/skills` and copied into `plugins/documents` through `lib/artifacts/skills/mapping.json`.
